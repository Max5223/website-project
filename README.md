# Командный веб-сайт

## Описание проекта
Совместный проект, разработанный командой из трех человек. Каждый участник отвечал за свою часть:

### Участники:
1. **Дизайнер/Архитектор** - отвечал за общую структуру, навигацию и главную страницу
2. **Участник 2** - разработал персональную страницу с интерактивными элементами
3. **Участник 3** - создал страницу с анимированными прогресс-барами

## Технологии
- HTML5
- CSS3
- JavaScript (ES6+)
